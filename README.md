# JSON関数を使って外部サービスの情報を取得しよう
- バージョン16で登場した新しいJSON関数について、どのような使い方が出来るかデモを通してご紹介します。外部サービスと連携し、カスタム Appをより便利に活用しましょう。
– 住所から緯度経度を取得する
– 2つの緯度経度から距離を取得する
– 外部サービスからJSONデータを取得する方法
– 取得したJSONデータをフィールドに設定する
– 取得したJSONデータからレコードを作成する
– その他活用方法など
